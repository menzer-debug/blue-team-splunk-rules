
# Blue Team Splunk Detection Rules

This repository contains SIEM detection rules developed for monitoring web application attacks using Splunk.

## Features
- OWASP Top 10 based detection rules
- Detection of common web attacks (XSS, SQL Injection, Command Injection)
- Real-time alerting support
- Integration with Splunk SIEM

## Structure
- splunk_rules/ → Splunk SPL rules
- sigma_rules/ → Sigma rules (optional)

## Usage
These rules can be deployed into Splunk for real-time monitoring and alert generation.
